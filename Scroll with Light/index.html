Scroll with Light Effect

Welcome to the Light Show

This interactive demonstration shows how you can create an engaging scrolling experience with a spotlight effect that follows your cursor and scroll position.

As you move down the page, watch how the light reveals content and creates a dramatic atmosphere.

How It Works

The spotlight effect is created using a radial gradient with the mix-blend-mode property set to "screen". This creates a light-like effect that brightens content underneath.

We track both scroll position and mouse movement to position the light dynamically, creating an interactive experience that responds to user actions.

Customizable Options

Try adjusting the controls in the bottom-right corner to customize the light effect. You can change:

- Light size
- Light intensity
- Light color
- Particle effects

Each adjustment creates a different mood and visual impact.

Practical Applications

This effect can be used for:

- Creative portfolios
- Product showcases
- Storytelling websites
- Interactive presentations

It draws attention to specific content elements and creates a sense of discovery as users navigate through the page.

Building Your Own

To create this effect, you'll need to understand:

- CSS positioning and gradients
- JavaScript event listeners
- The Intersection Observer API
- Browser performance optimization

The code is structured to be easily adaptable for your own projects.

Performance Considerations

While visually impressive, this effect needs to be implemented carefully to ensure good performance.

We're using requestAnimationFrame for smooth animations and limiting how frequently we update the position to avoid performance issues on lower-powered devices.
